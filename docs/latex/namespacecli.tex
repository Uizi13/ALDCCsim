{\footnotesize\ttfamily \label{namespacecli_a12ec60634710a096b3d2f50ecead7644} 
\doxymbox{\hyperlink{structcli_1_1_input}{Input}} cli::\+parse (\begin{DoxyParamCaption}\item[{int}]{argc}{, }\item[{char \texorpdfstring{$\ast$}{*}}]{argv}{\+[$\,$]\+}\end{DoxyParamCaption})}



Parses the command line arguments to extract input file, output file, and precision. 

It checks for the \textquotesingle{}-\/i\textquotesingle{}, \textquotesingle{}-\/o\textquotesingle{}, and \textquotesingle{}-\/p\textquotesingle{} switches and reads the subsequent argument. It also prints usage instructions if no arguments are provided. Throws a std::\+runtime\+\_\+error if a switch is expected but not found, or if the required input/\+output filenames are missing.
