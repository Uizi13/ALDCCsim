Implements the bisection method to find the variable element\textquotesingle{}s value that yields the desired output value. 

It iteratively narrows the search interval ({\ttfamily range\+[0]\+} and {\ttfamily range\+[1]\+}) based on the sign of the difference between the current output and the desired output, taking into account the derivative sign. The iteration runs for \textquotesingle{}constants::\+\+BISECTION\+\_\+\+PREC\textquotesingle{} steps.
